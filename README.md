

# GEAR-GUARD

### *Empowering Maintenance, Elevating Efficiency, Ensuring Reliability*

Gear-Guard is an open-source maintenance management platform that combines real-time simulation, secure backend infrastructure, and a modern frontend architecture to streamline asset tracking, scheduling, and workflow validation. Designed for developers, it facilitates testing, validation, and deployment of robust maintenance systems.

## Modern Development Stack

Gear-Guard is built with a high-performance stack for rapid, scalable development:

* **Frontend:** React, Vite, Tailwind CSS
* **Backend:** Express, Node.js, Socket.io
* **Database:** PostgreSQL, Mongoose
* **Tooling:** ESLint, Axios, PostCSS, Autoprefixer

## Getting Started

### Prerequisites

This project requires the following dependencies:

* **Programming Language:** JavaScript
* **Package Manager:** npm

### Installation

Build Gear-Guard from the source and install dependencies:

1. **Clone the repository:**
```bash
git clone https://github.com/dharmi10/gear-guard

```


2. **Navigate to the project directory:**
```bash
cd gear-guard

```


3. **Install the dependencies:**
```bash
npm install

```



---

## Usage

To run the project in a local development environment:
